# 🌐 글로벌 테크 뉴스 — 작전 상황실 (Tactical Globe)

[brswinvesting 뉴스 타임라인](https://brswinvesting.phyksw.workers.dev/news)의 156건(AI·반도체·우주·방산·양자·자율주행)을
**다크 벡터 지구본 + 클린 인포그래픽 대시보드**로 시각화한 인터랙티브 HTML입니다.

## 실행
`index.html`을 **더블클릭**하면 끝입니다. 서버·설치·인터넷 불필요(라이브러리·뉴스·국가지도를 모두 로컬에 포함, 오프라인 동작).
> 로컬 서버로도 가능: `node server.js` → http://localhost:8123

## 디자인
- **작전 상황실 지구본**: 어두운 바다 + 대륙 벡터 면 + **진한 국경 윤곽선** · 위경도 그리드 · 블루 대기광 · **멀리 공전하는 달** · 코너 브래킷·스캔라인·중앙 레티클 HUD · 실시간 LAT/LON/TGT readout.
- **거래선**: 굵은 섹터색 아치가 출발→도착으로 **흐르는 대시 + 방향 화살표**. 섹터·국가·뉴스를 고르면 관련 연결선이 자동으로 뜨고, 상단 `거래선` 토글로 전체(27선) 보기.
- **한국 지도**: 제주도·독도를 KR 영토로 주입 → 한국과 함께 하이라이트(독도는 라벨도 표시). *(110m 지도엔 둘 다 빠져 있어 별도 주입)*
- **인포그래픽 칩**: 섹터 칩 = 분야별 대표 아이콘 타일(AI·반도체·우주·방산·양자·자율주행·데이터센터·기타). 종류 칩도 아이콘.
- **폰트**: HUD·수치 `Chakra Petch`(테크 모노), 한글 본문 `Pretendard`. 여백·둥근 카드의 깔끔한 정보 위계.

## 기능 (요청 → 구현)
| 요청 | 구현 |
|---|---|
| 뉴스 국가를 핀으로 | 각 뉴스를 주체 기업 본사 도시에 섹터색 핀으로 산포(하이라이트 영토 **위**에 표시) |
| 뉴스 누르면 회전 | 카드/핀 클릭 → 해당국 회전 + 펄스 링 + 본문 펼침(HUD `TARGET LOCK`) + 그 뉴스의 거래선 |
| 필터 누르면 영토 하이라이트 | 섹터·종류·국가 칩 → 해당국 영토를 우세 섹터색으로 발광 |
| 섹터별 강세 | 인포그래픽 섹터 막대 + 국가 포커스(그 나라 강세 섹터) + 국가 순위 |
| 섹터 선택 → 국가 연결선 | 섹터를 고르면 그 섹터의 국가 간 거래선이 자동으로 이어짐 |
| 국가 선택(섹터 없음) → 모든 연관선 | 그 국가가 얽힌 모든 섹터의 거래선을 연결 |

## 국가 매핑 기준
원본 뉴스에 국가 필드가 없어 **헤드라인 주체 기업의 본사 국가**로 매핑.
예: 테슬라·SpaceX·OpenAI→🇺🇸, 삼성·SK·한화·KAI→🇰🇷, TSMC→🇹🇼, Alibaba·Zhipu→🇨🇳, 모빌아이→🇮🇱,
Nebius→🇳🇱, 라인메탈→🇩🇪, 페라리→🇮🇹, TCS→🇮🇳, D-Wave→🇨🇦, 드론실드→🇦🇺.
분포: **US 128 · KR 13 · CN 5 · 그 외 9개국** (156건 / 12개국). 규칙·예외는 `enrich.js`에서 수정.

## 파일 구조
```
index.html              앱 진입점 (더블클릭)
style.css               스타일 (작전상황실 다크 + 인포그래픽 + HUD)
app.js                  앱 로직 (globe.gl 제어·달/거래선·필터·타임라인·HUD)
vendor/globe.gl.min.js  지구본 라이브러리 (three.js r183 내장)
data/news.js            뉴스 139건 (window.NEWS_DATA)        ← 빌드 산출물
data/world.js           국가 영토 GeoJSON (window.WORLD_GEOJSON) ← 빌드 산출물
─ 데이터 파이프라인 (재생성용) ─
news_raw.html · parse.js · enrich.js · build-data.js · data/countries.geojson
```

## 데이터 갱신
```
node build-data.js --fetch   # 라이브 뉴스 재다운로드 → data/*.js 재생성
node build-data.js           # 받아둔 news_raw.html로 재생성만
```
`--fetch`는 cheerio 필요: `npm i`
